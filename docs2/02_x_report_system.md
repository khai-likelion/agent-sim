# 🧠 X-Report 분석 시스템

## 1. 개요
X-Report는 특정 매장의 현황을 진단하고 성공적인 운영을 위한 전략적 솔루션을 제안하는 리포트입니다. OpenAI의 최신 모델인 **GPT-5.2**를 활용하여 데이터 사이언티스트 수준의 심층 분석을 제공합니다.

## 2. 분석 프로세스 (`report_engine.py`)

### Step 1: 심층 데이터 분석 (Analytic Logic)
- **Persona**: 전문 데이터 분석가 및 전략 컨설턴트 페르소나 부여.
- **Input Data**:
    - 매장 기본 정보 (메뉴, 가격대, 위치)
    - 리뷰 메트릭스 (맛, 서비스, 청결도 등)
    - 상권 인구 통계 데이터 (유동/생활 인구 구성비)
    - 경쟁 매장 맥락 정보
- **Temperature**: 0.1 (분석의 정확성과 일관성을 위해 낮게 설정)

### Step 2: 리포트 생성 및 시각화 (Visualization)
- **Format**: Markdown 기반의 구조화된 보고서.
- **주요 섹션**:
    - 매장 핵심 스코어보드 (상권 평균 대비 성능)
    - SWOT 분석 (강점, 약점, 기회, 위협)
    - 타겟 고객 페르소나 정의
    - **실행 가능한 솔루션 (Actionable Solutions)**: 시뮬레이션에 바로 적용할 수 있는 구체적 제안.

## 3. 기술적 특징 및 개선 사항
- **GPT-5.2 최적화**: `.env` 설정을 통해 최신 모델을 안정적으로 호출.
- **유니코드 안정성**: 윈도우 터미널 환경에서도 에러 없이 기호 및 한글을 출력할 수 있도록 인코딩 이슈 해결.
- **구조화된 저장**: 분석 결과는 `_result.json`으로, 가독성 높은 보고서는 `_report.md`로 분리 저장하여 데이터와 문서를 동시 관리.
- **Frontend 연동**: `@tailwindcss/typography`의 `prose` 설정을 통해 웹 대시보드에서 전문 잡지 수준의 레이아웃으로 렌더링.

## 4. 데이터 소스 경로
- **Raw Data**: `agent-sim/data/raw/split_by_store_id_ver3/`
- **Generated Reports**: `generated_reports/`
